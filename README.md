# Vue 元件化
- 使用 VeeValidate 套件完成表單驗證功能，操作文件（無ESM 版本）
- 使用 loading 套件製作 loading 效果
- 串接前台 API 完成購物車功能

### 注意：
- 新增相同產品到購物車時需累加項目
- 送出訂單後，購物車需要清除原本項目
- 購物車無產品時不建議發出結帳請求
